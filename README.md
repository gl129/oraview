<H1>OraView</H1>

<h2>Oracle database DBA tool:</h2>

- Active Session History viewer
- (todo) Session control
- (todo) Statistics viewer


Features:
- tnsnames file reader
- edit and save connections
- stacked waitclass graph, selectable time interval, AWR reporting
- database load grouping and filtering by session/user/sql/object/wait event/service
- expandable raw ASH viewer, column hide and size adjustement
- precache users/object/service names and sql texts
- on id-mouseover user/object/service names and sql text popups
